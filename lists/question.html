Block Elements

Block elements are those elements which always start from next line

Examples :

Ordered list element :

  1. h1 tag
  2. h2 tag
  3. p tag

Inline Elements

Inlines elements are those elements which start from next point where previous element ended

Unordered list element :

  • audio
  • video
  • img
